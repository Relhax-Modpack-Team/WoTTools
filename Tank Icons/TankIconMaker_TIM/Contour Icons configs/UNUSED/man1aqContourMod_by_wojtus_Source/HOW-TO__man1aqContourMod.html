This will be a long email, so please take your time. It's not so complicated once you're inside, however I prefer to describe it in detail now.

Let me know if you have any questions/doubts/troubles.


First, the downloads for you are here: http://d.wojtus.net/123GAUSS/ - please download them before reading further.


Here are the files for man1aqContourMod:

1. As you might suppose, I'm using Tank Icon Maker to manage the mod and this tool does 95% of the work. You can download TIM from koreanrandom forum, but since there were few forks of the project, I'm not sure all of the are up-to-date. The most recent problem was WG's switch from PNG maps to .DDS atlases. In TIM.RAR you'll find the binaries I've been using successfully if you don't want to download. ----------^^^^^^^------------------ ignore this step 1. TIM.rar is not actual anymore

2. TIM_Data_CSVs.rar content has to be placed in TIM/Data directory. The make_builtin-2.bat file should be run every game update, sometimes manual corrections to file WotBuiltIn-3.csv are required (you'll get the idea once looking into files)

3. man1aqContourMod-source.rar  - this is the original file I've got from man1aq - he didn't used TIM at the time. Most valuables inside are original Photoshop templates, however the templates I currently use are in PSD.RAR file and that's actually the only one you need. Make sure you have font Microsoft PhagsPa installed in the system. If not, it's here.

4. color_templates.psd is the file I've used for creating TIM templates - you won't need to use it normally.

5. [MAIN]names.psd is the Photoshop file where new tanks names are made.

6. TIM_style.xml is the backup of TIM settings. You should import this into your setup.

7. Now, mr.rar are the main mod files used by TIM. I have it as "mr" subdirectory in TIM installation. If you'd like to have it somewhere else, you need to change paths in the settings imported in point 6.

8. Install WinRAR (C:\Program Files) if you'd like to use my automation script.


And now the instructions - the full workflow of an update, once you have everything in place:

1. Update game client first.

2. Run Data/make_builtin-2.bat in TIM directory - this can be skipped for micropatches etc.

3. Check if any changes to WotBuiltIn-3.csv are required (well, you'll usually spot it much late than now).

4. Start TankIconMaker.exe It should look like on the screenshot attached.

5. Check if it shows correct game version in top-left corner.

6. Check for any icons which have a "warning triangle" by it. This means there's no custom file for this tank. It's ok for tanks which are absent from game, like IS-3A (T), Jagdtiger (H), T95/Chieftain, however it's good practice to create them, because usually the icon is in the game a few months before the tank is actually introduced.

If no icons are missing (micropatch etc.), skip to point 17.

7. Open [MAIN]names.psd in Photoshop.

8. Find a tank with a similar length of name and use as a template - or create it from the scratch. This should be easy.

9. The guidelines from man1aq which I've followed were:

  • Use full tank names if possible.
  • Do not make differences for WG skinned tanks, like Scorpion and Scorpion G. If tank stats are the same, it should be same icon.
  • Make the name which you expect the tank to be called by players the most prominent.
  • Make a heavy use of font kerning and vertical interline. The text should as big as possible.

10. Select proper background colour layer for the text per tank class (layers at the bottom of list).

11. Switch to TIM and hover the mouse over it to see the internal file name. It's like "germany-G79_Pz_IV_AusfGH". Unfortunately there's no way to copy-paste it.

12. In Photoshop, use Save-For-Web and save it as PNG24, no transparency, no interlacing. Use the name from previous point with .png extension.

13. Place this new PNG file under per_tank/text directory.

14. Repeat for every missing icon. :)

15. Once done, simply reselecting any layer in TIM (F3 selects/deselects layers) should reload the icons.

16. Use "TEST type/colors" layer to cross-check if you didn't make mistake in background colour. Or if WG changed tank category.

17. Once all icons are here, it's time to build mod files. First you should set the layers for every mod flavour. For the basic (plain), the layers to be enabled are: background, 1. PLAIN, default text, handcrafted text, special files.

18. At the bottom, click three dots next to Path: (standard). The first two options (Normal icons and Battle atlas) should be enabled with no paths given. This setting is saved, you only need to check it once.

18. Now click "Save as" at the bottom and use last option "Bulk save to folder" and select dist/_plain folder (do not descend into folder, just select it). Click "Save" in next dialog. After a few seconds TIM returns to main screen.

19. For next mod flavours, you have to switch layers accordingly. There are labelled to make this easier of course. Only switch those with numbers. So:

  • Premium alternative - use layers 2 and 3 and save to premalt
  • Unique premium - use layer 3 and save to unique_premium
  • Violet - use layer 7 and save to violet
  • Fun - use layers 1 and 8 and save to fun (it's ok many warning triangles will appear)

20. As a good practice, deselect layer 8 to get back to plain mode. Easy to miss it next week!

21. Layers 4-6 are from flavours I've deprecated already. It looked like nobody was using them. Use your own wisdom regarding flavours.

22. Once all 5 flavours are saved, you may close TIM. Open mod dist/ directory.

23. Edit the meta_xml.template accordingly to your liking. This will be put as meta.xml in all .wotmod files.

24. Run make_distro.bat script. It asks for patch number. Default is 0, so final mod version will be eg. 1.5.0.4_p0. Then it creates all files using WinRAR. The .bat works fine with my Windows 7, I haven't tested it with Win10 thou!

25. In dist/.check directory there are PNG atlases. You can have a look at them to make sure you didn't make any mistakes with layers in TIM.

26. New .ZIP files appear in the dist/  directory, these are the ones you may publish on FTP.

27. dist/.wotmod directory now contains .wotmod packages, which can be used on wgmods.net and probably will be used in your modpack.


DONE! :)
